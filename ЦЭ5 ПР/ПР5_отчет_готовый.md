# Практическая работа №5
**Дисциплина:** «Цифровая экономика и процессное управление предприятием»  
**Тема:** «Знакомство с платформой 1С:Предприятие»  
**Вариант:** 1 — Розничный магазин одежды

## 1. Задание на занятии (вариант 1)
### 1.1 Проблемы «до автоматизации»
- Отчёты из офлайн и онлайн торговли не совпадают.
- Возвраты с маркетплейсов сложно отслеживать.
- Программа лояльности ведётся вручную.
- Много времени уходит на сверку остатков.

### 1.2 Процессы для автоматизации
- Единый учёт продаж по всем каналам.
- Учёт и обработка возвратов.
- Управление программой лояльности.
- Сквозной складской учёт.
- Автоматизированная управленческая и регламентированная отчётность.

### 1.3 Предлагаемое решение
- Базовая конфигурация: 1С:УНФ (облачный вариант).
- Дополнительно: модуль обмена с маркетплейсами, интеграция с сайтом, CRM-блок.

### 1.4 Необходимые интеграции
- Сайт интернет-магазина (обмен заказами и остатками).
- Маркетплейсы (заказы, возвраты, статусы).
- Онлайн-кассы.
- CRM/BI (для расширенной аналитики).

### 1.5 Ключевые функции и отчёты
- Функции: единый каталог номенклатуры, заказы, продажи, возвраты, остатки, лояльность.
- Отчёты: продажи по каналам, возвраты, остатки по складам, маржинальность, эффективность акций.

### 1.6 Структура доступа
- Администратор: полный доступ.
- Менеджер продаж: заказы, клиенты, лояльность.
- Кладовщик: складские документы и остатки.
- Руководитель: отчёты и аналитика (без изменения нормативно-справочной информации).

### 1.7 Краткое обоснование для «директора»
Предложенный подход устраняет разрозненность данных между офлайн и онлайн каналами, сокращает объём ручных операций и предоставляет актуальную отчётность в облачном формате без собственного сервера.
